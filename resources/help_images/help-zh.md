# 登录

## 登录

![](direct/help_login_page.png)

连接宝宝管家服务器有两种方式：
输入您的宝宝管家服务器地址、用户名和密码，然后点击登录按钮将应用连接到服务器。
或者选择二维码登录方式（向下滚动查看详情）。

## 二维码登录

![](direct/help_login_qrcode.png)

您也可以使用登录二维码登录宝宝管家服务器。请使用其他设备登录宝宝管家服务器，
然后通过用户菜单（右上角）进入"添加设备"页面，找到登录二维码。
用手机上的应用扫描二维码。扫描成功后，点击"立即登录"按钮即可登录。

## 不安全登录

![](screenshots/help_insecure_http_warning.png)

如果您使用的是 http 地址而不是 https 地址，应用会显示此警告。
请仅在您自己控制的网络内对本地宝宝管家服务器使用此功能！

## Home Assistant Ingress

![](direct/help_login_homeassistant.png)

宝宝管家 Android 版部分支持登录托管在 Homeassistant Ingress 服务器上的宝宝管家服务器。
您需要使用二维码登录方式登录此类服务器。

# 概览

## 概览

![](screenshots/help_overview_2.png)

登录后，您可以在屏幕上方找到记录活动的按钮，
屏幕下方显示最近记录的活动历史。
如果您为多个孩子记录活动，可以左右滑动在孩子之间切换。

# 使用计时器记录

## 使用计时器

![](direct/help_play_button_highlight.png)

要启动计时器，请点击屏幕上方的计时器事件按钮之一。
这将启动一个计时器。

## 使用计时器

![](screenshots/help_timer_interface.png)

启动计时器后会打开与该计时器相关的界面。
填写您想要记录的所有信息。当所有必填字段都填写完毕后，会出现保存按钮。

## 使用计时器

![](direct/help_save_button_highlight.png)

保存按钮出现后，您可以点击保存按钮停止计时器。
您也可以再次点击顶部的计时器按钮来取消计时器，而不是保存。

# 使用事件记录

## 记录尿布和笔记

![](screenshots/help_diaper_logging.png)

除了计时器事件，顶部的按钮行还允许记录非计时器事件，
如尿布更换和通用笔记。控制方式与计时器记录相同，但没有计时器功能。

# 事件历史

## 事件历史

![](screenshots/help_events.png)

屏幕底部的事件历史显示所选孩子的最近活动记录。
您可以长按历史中的条目三秒钟来编辑条目。
这将在宝宝管家应用中打开相应的页面。
要删除条目，请点击红色的"删除"按钮。
