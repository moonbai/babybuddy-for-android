# 安卓端 Baby Buddy
本应用必须搭配**已正常运行的 Baby Buddy 服务端**才可使用。
若你没有搭建并运行 Baby Buddy 服务端，该客户端将无法正常使用。

如需搭建 Baby Buddy 服务端，可使用简易部署方案，一键部署至 Heroku：
[Heroku 一键部署 Baby Buddy](https://elements.heroku.com/buttons/cdubz/babybuddy)
